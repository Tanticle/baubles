# Baubles 2

Baubles 2 is a from-scratch continuation of the original [Baubles by Azanor13](https://www.curseforge.com/minecraft/mc-mods/baubles) made for modern Minecraft.

## Gradle

[CurseMaven](https://cursemaven.com/) is recommended when adding Baubles 2 to your project. The dependency string for the latest version (Minecraft 1.21.4, Baubles 2 Pre-Release 0.6) is:
> **curse.maven:baubles-2-1040353:7588727**
